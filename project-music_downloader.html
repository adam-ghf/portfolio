Quick Mode

Dit is één van de drie opties. Quick Mode wordt gebruikt om muziek te taggen binnen dezelfde categorie.

Je begint met dit venster. Hier kun je de video, afbeelding, titel, artiest en genre aanpassen.

Je kunt ook de knop "Extract" gebruiken om de informatie opnieuw in te vullen als je iets fout hebt geschreven.

Als je klaar bent kan je verder gaan naar de "category" window verder gaan

Category

Eens je je informatie hebt ingevoerd, moet je een categorie kiezen. Je selecteert er één, en die wordt naar de server gestuurd om gedownload te worden.

Advanced Mode

In Advanced Mode krijg je veel meer tags om uit te kiezen, zoals album, albumartiest, schijfnummer en tracknummer. Het tracknummer wordt automatisch overgenomen van het playlistnummer, dus je hoeft dit niet elke keer aan te passen.

In deze mode is er geen categorievenster; je stuurt de gegevens rechtstreeks naar de server.

Preset Mode

Deze mode werkt exact zoals Advanced Mode, maar is iets unieker. Hij is bedoeld om de informatie te bewaren, zelfs na het aanpassen of vernieuwen van de webpagina. Dit is erg handig als je veel nummers van hetzelfde album of dezelfde artiest downloadt.

Debug Window

Het debugvenster geeft allerlei informatie, zoals wat er is geëxtraheerd, eventuele fouten en de uiteindelijke string die naar de server wordt gestuurd. Het is erg handig als je zeker wilt zijn dat het script correct werkt.
